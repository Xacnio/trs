# trs
A simple translation tool on command line

# Installation
```
> go install github.com/xacnio/trs
> trs
```

# Preview 
```
auto » en

Source Text:
Target Text:


Cmds: s <source language> / t <target language> / <source text>
>
```
# Example
```
> s en
> t tr
> Hello
```
**Screen:**
```
en » tr

Source Text: Hello
Target Text: Merhaba


Cmds: s <source language> / t <target language> / <source text>
>
```
